# Data Structure
Several commonly used data structures are implemented with C++11.

- AVL tree
- binary heap
- binary tree array
- binary tree list
- graph adjacency matrix
- hash map list
- red black tree
- ring queue
- sequence list
- single list
- skip list
- stack

# Compile
## Environment
- OS: Linux
- Cmake Version: >= 3.8
- GCC Version: >= 4.8

## Clone
```
git clone https://github.com/Bruce-Lee-LY/data_structure.git
```

## Build
```
cd data_structure
./build.sh -t Release -b OFF
./build.sh -t Debug -b ON
```

# Run Sample
```
./run_sample.sh
```
